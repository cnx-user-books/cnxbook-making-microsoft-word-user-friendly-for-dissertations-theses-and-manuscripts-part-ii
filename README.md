# Making Microsoft Word User-Friendly for Dissertations, Theses, and Manuscripts: Part II

[![Open in Gitpod](https://gitpod.io/button/open-in-gitpod.svg)](https://gitpod.io/from-referrer/)

_Making Microsoft Word User-Friendly for Dissertations, Theses, and Manuscripts: Part II_ is a textbook published by [OpenStax](https://openstax.org/), a non profit organization that is part of [Rice University](https://www.rice.edu/).

The book can be viewed [online](https://github.com/cnx-user-books/cnxbook-making-microsoft-word-user-friendly-for-dissertations-theses-and-manuscripts-part-ii/releases/latest), where you can also see a list of contributors.
